# V-й Международный и I-й Виртуальный конгресс по эндоскопической гинекологии

**City:** Общая информация

21-25 сентября 2020г. состоится V-й Международный и I-й Виртуальный конгресс по эндоскопической гинекологии.
Конгресс будет проходить в онлайн-формате на платформах https://endosurgery.pro/ и Zoom.
На онлайн-площадке соберутся вместе ведущие гинекологи-эндоскописты Казахстана, Франции, Италии, Индии, России, Украины, Азербайджана для обмена своим накопленным опытом, покажут живую хирургию из операционных разных городов и стран.
Основные темы:
21.09 - гистероскопия/гистерорезектоскопия, репродуктивная гинекология;
22.09 - эндометриоз;
23.09 - пролапс;
24.09 - малоинвазивная хирургия в онкогинекологии;
25.09 - реконструктивная хирургия органов малого таза.
Поскольку конгресс будет проходить в онлайн-формате в программе возможны небольшие изменения, связанные с техническими возможностями интернет-подключений разных стран. О возможных изменениях информация будет размещаться на сайте.
Участникам Конгресса будут доступны презентации и видео-кейсы в закрытом доступе для просмотра в удобное время.
Для участия в конгрессе необходимо пройти ОБЯЗАТЕЛЬНУЮ регистрацию https://endosurgery.pro/register/
Доступ к участию возможен только по уникальной ссылке, которая будет выслана на подтвержденный адрес электронной почты.
Приглашаем всех желающих принять участие в Конгрессе!
