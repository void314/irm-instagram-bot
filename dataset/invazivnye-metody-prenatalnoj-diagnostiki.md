# Инвазивные методы пренатальной диагностики

**Date:** 2017-11-20 15:52:24
**City:** Общая информация
**Type:** page
**Link:** https://irm.kz/vedeniye-beremennosti/invazivnye-metody-prenatalnoj-diagnostiki/

---

**Инвазивные методы пренатальной диагностики –** **внутриматочные** **операции под ультразвуковым контролем с целью получения плодного материала (хорионбиопсия, амниоцентез, кордоцентез) для проведения пренатального кариотипирования при подозрении на хромосомную патологию плода.**
Инвазивные методы исследования проводятся по назначению врача-генетика с целью получения плодного материала для выявления хромосомных болезней, носительства хромосомных аномалий, а также моногенных болезней.
**Стандартные показания для направления на инвазивную диагностику, одобренные ВОЗ, включают:**

* Возраст беременной старше 35 лет
* Наличие не менее 2-х самопроизвольных абортов на ранних сроках беременности
* Наличие в семье ребенка, или выявление при предыдущей беременности плода с синдромом Дауна
* Множественные врожденные пороки
* Семейное носительство хромосомных перестроек
* Результаты неинвазивных тестов с выявленным высоким риском хромосомной патологии
* Биохимические или ультразвуковые маркеры

Выбор инвазивного метода определяется сроком беременности, показаниями к его проведению, условием доступа и т.д.
Сроки проведения – с 9-й недели до 21-ю недели.
